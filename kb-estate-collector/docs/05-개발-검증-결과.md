## 4. 버그 수정 기록

### 4-1. Boolean 임포트 누락 (치명적)

**파일**: `src/models/price_data.py`

**증상**:
```
NameError: name 'Boolean' is not defined
→ Table 'kb_prices' is already defined for this MetaData instance
→ 6개 모듈 연쇄 임포트 실패
```

**원인**: SQLAlchemy `Boolean` 타입을 사용하지만 import 하지 않음
```python
# 수정 전
from sqlalchemy import Column, Integer, String, DateTime, Float, Date, ForeignKey, Text, Index, Enum

# 수정 후
from sqlalchemy import Column, Integer, String, DateTime, Float, Date, ForeignKey, Text, Index, Enum, Boolean
```

**영향 범위**: `price_data.py` → `crawl.py` → `tasks.py` → `jobs.py` → `complexes.py` → `complex_discovery.py` (6개 모듈 연쇄 실패)

**교훈**: SQLAlchemy 모듈이 부분적으로 로드된 후 실패하면, 이미 등록된 테이블 메타데이터 때문에 재임포트 시 별도의 에러가 발생. 근본 원인 추적이 어려움.
