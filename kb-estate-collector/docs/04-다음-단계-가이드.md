## 즉시 해야 할 것

### 1단계: Docker 통합 테스트

```bash
# PostgreSQL + Redis + API + Worker 전체 시작
docker-compose up -d

# 로그 확인
docker-compose logs -f

# DB만 먼저 시작 (로컬 개발 시)
docker-compose up -d postgres redis
```

### 2단계: DB 마이그레이션

```bash
# Alembic 마이그레이션 생성 (KB_TRANSACTION JobType 추가 등)
alembic revision --autogenerate -m "Add KB_TRANSACTION job type and browser config"

# 마이그레이션 적용
alembic upgrade head
```

### 3단계: 단지 발견 테스트

```bash
# 강남구(11680) 아파트 단지 자동 발견
curl -X POST "http://localhost:8000/api/complexes/discover-region?region_code=11680"
```

예상 결과:
```json
{
    "region_code": "11680",
    "total_found": 258,
    "new_registered": 258,
    "already_exists": 0,
    "complexes": [{"name": "거평프리젠", "kb_complex_id": "12", "is_new": true}]
}
```

### 4단계: 시세 수집 테스트

```bash
# Job 생성 후 실행
curl -X POST http://localhost:8000/api/jobs \
  -H "Content-Type: application/json" \
  -d '{"name": "KB시세수집", "job_type": "kb_price", "is_active": true}'

curl -X POST http://localhost:8000/api/jobs/1/run

# 또는 지역 전체 수집 (발견→시세→실거래가→매물)
curl -X POST "http://localhost:8000/api/jobs/run-region?region_code=11680"
```

### 5단계: 수집 데이터 확인

```bash
# KB시세 조회
curl "http://localhost:8000/api/data/kb-prices?complex_id=1"

# CSV 내보내기
curl "http://localhost:8000/api/data/kb-prices/export?complex_id=1" > prices.csv

# 실거래가/매물 조회
curl "http://localhost:8000/api/data/transactions?complex_id=1"
curl "http://localhost:8000/api/data/listings?complex_id=1"
```

### 6단계: 매물 API 파라미터 확인

매물 API(`propList/stutCdFilter`)의 정확한 파라미터 조합 확인 필요.

## KB API 직접 테스트 (DB 없이)

```bash
# 시세 API 직접 호출
python -c "
import httpx, json
r = httpx.get('https://api.kbland.kr/land-price/price/BasePrcInfoNew',
    params={'단지기본일련번호': '12', '면적일련번호': '127753'},
    headers={'Referer': 'https://kbland.kr', 'Origin': 'https://kbland.kr'})
data = r.json()
sise = data['dataBody']['data']['시세'][0]
print(f'일반가: {sise[\"매매일반거래가\"]:,}만원')
print(f'상한가: {sise[\"매매상한가\"]:,}만원')
print(f'하한가: {sise[\"매매하한가\"]:,}만원')
print(f'기준일: {sise[\"시세기준년월일\"]}')
"

# 단지 검색 API 직접 호출
python -c "
import httpx, json
r = httpx.post('https://api.kbland.kr/land-complex/map/map250mBlwInfoList',
    json={'물건종류': '01', 'startLat': 37.48, 'startLng': 127.01,
          'endLat': 37.54, 'endLng': 127.07, '거래유형': '1,2,3',
          'selectCode': '1,2,3', 'zoomLevel': 14},
    headers={'Referer': 'https://kbland.kr', 'Origin': 'https://kbland.kr'})
data = r.json()['dataBody']['data']
complexes = data.get('단지리스트', data.get('list', []))
print(f'발견된 단지: {len(complexes)}개')
for c in complexes[:5]:
    print(f'  {c.get(\"단지명\", \"?\")} (ID: {c.get(\"단지기본일련번호\", \"?\")})')
"
```
